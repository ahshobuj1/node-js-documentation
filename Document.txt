Node Modules

    -> Module is set of function. (think about js libraries such as math).
    ->There are 3 types of Node module .
        1. Local Modules (Own created Modules)
        2. Built-in Modules (Node.js own Modules) - http, url,patch, fs .
        3. External Modules (Managed by npm)(Third party Modules)
